# DevOps Quiz: Questions & Answers

## Git & Jenkins
1. Which Git command is used to apply changes from one branch into another, without creating a new commit?
   - Answer: git rebase

2. In Jenkins, which file defines the pipeline as code?
   - Answer: Jenkinsfile

3. In Jenkins, what is a 'Freestyle Project' used for?
   - Answer: A simple build job

4. What is the role of Jenkins in the CI/CD pipeline?
   - Answer: Automating the build, test, and deployment process

5. Which Git command is used to synchronize the local repository with the remote one without making any changes to the working directory?
   - Answer: git fetch

6. In Git, what does the 'git cherry-pick' command do?
   - Answer: Apply a commit from one branch onto another

## Docker & Kubernetes
7. In Docker, what does the command 'docker-compose up' do?
   - Answer: Starts and runs containers defined in docker-compose.yml

8. What does the Kubernetes 'kubectl' command-line tool allow you to do?
   - Answer: Manage Kubernetes clusters

9. Which of the following is not a type of service in Kubernetes?
   - Answer: ReplicaSet

10. What is the purpose of 'ConfigMap' in Kubernetes?
    - Answer: Store non-confidential configuration data

11. In Docker, what is a 'Volume' used for?
    - Answer: Persisting data beyond the lifecycle of a container

12. Which Kubernetes object is responsible for scaling the number of pod replicas?
    - Answer: ReplicaSet

13. In Kubernetes, what is the purpose of a 'Namespace'?
    - Answer: Organize objects within the cluster

14. Which of the following Kubernetes components is responsible for managing the state of the cluster?
    - Answer: etcd

15. What is the role of 'Helm' in Kubernetes?
    - Answer: Package management

16. In Docker, what does the 'docker swarm' command enable?
    - Answer: Container orchestration

## Infrastructure as Code & Configuration Management
17. Which tool is most commonly used for Infrastructure as Code (IaC) in a multi-cloud environment?
    - Answer: Terraform

18. Which tool is best suited for performing automated configuration management?
    - Answer: Puppet

19. What is the main advantage of using Ansible in a DevOps environment?
    - Answer: Agentless architecture

## CI/CD, Microservices & Secrets Management
20. What is the primary purpose of a 'Service Mesh' in microservices architecture?
    - Answer: Managing service-to-service communication

21. Which of these is a popular CI/CD tool integrated with Kubernetes?
    - Answer: Spinnaker

22. Which of the following CI/CD tools supports both continuous integration and deployment?
    - Answer: GitLab CI

23. Which tool in the DevOps ecosystem is used for managing secrets and credentials in a secure way?
    - Answer: Vault

## ELK Stack (Logging & Monitoring)
24. Which component of ELK Stack is responsible for collecting log data from different sources?
    - Answer: Logstash

25. What is the primary function of Logstash in the ELK Stack?
    - Answer: Data collection and transformation
